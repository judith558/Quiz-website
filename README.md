# Quiz App

A simple, interactive quiz application built with HTML, CSS, and JavaScript. Test your knowledge across various topics with a timed quiz featuring multiple-choice questions.

## Features

- **13 Diverse Questions**: Covering sports, technology, animals, music, astronomy, and more.
- **Timed Quiz**: 60-second countdown timer to add excitement.
- **Interactive UI**: Radio button selections with visual feedback (green for correct, red for incorrect).
- **Score Tracking**: Displays final score at the end.
- **Restart Functionality**: Play again option to retry the quiz.
- **Mobile Responsive**: Optimized for mobile devices with adjusted layouts and touch-friendly buttons.
- **Start Menu**: Welcome screen to begin the quiz.

## Technologies Used

- **HTML5**: Structure of the application.
- **CSS3**: Styling, animations, and responsive design.
- **JavaScript**: Logic for quiz functionality, timer, and interactions.

## How to Run

1. Click "Start Quiz" to begin.
2. Select answers and submit to proceed through the questions.
3. View your score at the end and choose to play again.

## Project Structure

- `index.html`: Main HTML file containing the structure.
- `style.css`: CSS file for styling and responsiveness.
- `script.js`: JavaScript file handling quiz logic, timer, and events.
- `README.md`: This file.
